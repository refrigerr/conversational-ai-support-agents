# Setup Instructions

## Overview
Guide for installing and configuring the application.

## Prerequisites
- Node.js >= 18 or Python >= 3.10
- Package manager (npm, pip)
- Environment variables

## Installation
git clone <repo-url>
cd <project-name>
npm install

## Environment
APP_ENV=development
PORT=3000
DATABASE_URL=<db-url>
API_KEY=<api-key>

## Run
npm run dev

## Verification
- Open http://localhost:3000
- Check /health endpoint
